# DII-Global Europe

Landing page for **DII-Global Europe** — a Brussels-based non-profit (VZW / ASBL) working on information integrity, independent journalism and countering foreign information manipulation across the transatlantic space. Chaired by Vitaliy Sizov.

The European continuation of the Donetsk Institute of Information.

## Live site

Published via GitHub Pages: **https://alexmazuka.github.io/dii-global-europe/**

## Structure

- `index.html` — single-page landing (About · Focus · Programmes · Insights · Contact)
- `blog/before-february-2022.html` — first blog post: a Brussels research briefing
- `assets/img/` — briefing photography

## Flagship programme

The [Transatlantic Truth Lab](https://alexmazuka.github.io/transatlantic-truth-lab/) is DII-Global Europe's flagship analytical initiative.
